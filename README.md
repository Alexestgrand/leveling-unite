# Leveling: Unite – The Fragments

Événement communautaire Discord autour d'une grande énigme collective.  
**Site officiel :** [leveling-unite.vercel.app](https://leveling-unite.vercel.app)

---

## L'idée en bref

> **Aucun joueur ne peut gagner seul. Toute la communauté doit collaborer.**

Une **phrase secrète de 15 mots** est au cœur de l'enquête. Les indices arrivent par vagues, via les Fragmentés, les annonces Discord et TikTok.

Deux camps s'affrontent :
- la **Communauté Leveling** (les membres du serveur) ;
- le **Staff Leveling** (modération et administration Discord).

**L'organisateur** conçoit l'événement, désigne les Fragmentés et reste **neutre** — il ne joue dans aucun camp.

---

## Comment fonctionne l'énigme ?

### Les vagues

- **4 vagues** sur environ **20 jours** ;
- **5 énigmes par vague**, soit **1 vague tous les 5 jours** ;
- chaque énigme est confiée à un **Fragmenté** qui relaie des pistes à son camp ;
- les **indices publics** (annonces + TikTok) sont publiés sur le site.

### Les Fragmentés (lore du portail)

Tous les 5 jours, **5 Fragmentés** sont désignés **publiquement** par camp :

1. ils reçoivent une **énigme en message privé** ;
2. ils envoient la solution à l'organisateur pour validation ;
3. ils **ne publient jamais le mot en clair** ;
4. ils relaient des **indices indirects** dans le salon d'enquête de leur camp (type de mot, longueur, position, confirmation d'une théorie…) ;
5. en fin de vague, de **nouveaux** Fragmentés prennent le relais (rotation).

Les **Enquêteurs**, c'est tout le reste du camp : ils théorisent, mettent à jour le journal collectif, contribuent à TikTok et peuvent soumettre la phrase finale.

### La victoire

Quand un camp pense avoir reconstitué la phrase complète :

1. connexion Discord sur le site ;
2. soumission des **15 mots** sur la page **Soumettre** ;
3. **2 tentatives maximum par 24 heures** ;
4. le **premier camp** à obtenir une validation gagne.

La personne dont le compte valide la phrase en premier est le **Finder**.

## Déroulé sur Discord

### Salons principaux

| Salon | Rôle |
|-------|------|
| `#annonces-unite` | Lancement, listes de Fragmentés, paliers TikTok, victoire |
| `#journal-unite` | Fil épinglé — résumé de l'enquête (pour les arrivants) |
| `#enquete-membres` / `#enquete-staff` | Théories et débats du camp |
| `#fragmentes-membres` / `#fragmentes-staff` | Coordination entre les 5 Fragmentés |
| `#indices-publics` | Fragments stabilisés (indices meta, pas les mots bruts) |
| `#bienvenue-unite` | Guide pour rejoindre l'événement en cours de route |

### Une vague type (5 jours)

1. **Jour 1** — Annonce des 5 Fragmentés + début des énigmes en MP  
2. **Jours 2–4** — Énigmes suivantes, théories dans `#enquete`, mise à jour du journal  
3. **Jour 5** — Bilan de la vague, nouveaux Fragmentés annoncés pour la vague suivante  

### Arrivé en retard ?

Consulte dans l'ordre :
1. `#bienvenue-unite`
2. `#journal-unite` (épinglé)
3. ce site → **Indices**, **Déroulé**, **Règles**

Tu peux contribuer immédiatement sans relire tout l'historique du chat.

## TikTok

Les vidéos avec **#Levelingfitnessapp** et **#LevelingUnite** alimentent un objectif collectif de vues.

Les paliers débloquent des aides (lettre, catégorie de mot, retrait d'une fausse piste, indice décisif…).  
Les vues aident surtout le camp **Communauté** à rattraper le Staff si besoin.

## Récompenses

### Si la **Communauté** gagne

**20 Hunter Pass + 10 Nitro** (1 mois chacun) :

| Rôle | Lot |
|------|-----|
| **Finder** (1re phrase valide sur le site) | 2 Nitro + 1 Hunter Pass |
| **Top Fragmenté** (vote du camp) | 2 Nitro + 1 Hunter Pass |
| **Top Enquêteur** (vote du camp) | 1 Nitro + 1 Hunter Pass |
| **Giveaway** (participants actifs) | 5 Nitro + 17 Hunter Pass |

Pour les votes **Top Fragmenté** et **Top Enquêteur** : l'organisateur propose une shortlist de candidats ; le camp vote pour en désigner un.

### Si le **Staff** gagne

- **950 €** reversés à Leveling (futurs événements, projets communautaires) ;
- **5 Nitro** parmi les membres du staff ayant participé activement.

### Si personne ne trouve à temps

- **50 €** ajoutés à la cagnotte communautaire ;
- **2 Nitro** en tirage au sort ;
- la phrase est révélée officiellement.

---

## Règles essentielles

- **Pas de mot en clair** issu d'une énigme Fragmenté dans les salons publics.
- **Pas de capture d'écran** de messages privés contenant des fragments.
- **Pas de multi-comptes**, pas de collusion entre camps.
- **2 soumissions / 24 h** sur le site pour la phrase finale.
- En cas de **fuite** : le fragment est compromis, un indice de remplacement est publié, le fautif peut être exclu.
- L'**organisateur** tranche en dernier ressort.

Règles complètes sur le site : [leveling-unite.vercel.app/regles](https://leveling-unite.vercel.app/regles)

## Révélation finale

- déblocage d'un **salon secret** ;
- message du créateur de Leveling ;
- modalités de récupération des récompenses ;
- **rôle Discord exclusif** lié à l'édition The Fragments.
